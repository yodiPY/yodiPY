Hello, my name is **Yodi Dwi Cahyo**.
I am an **Informatics Engineering student at Universitas Pamulang** with an interest in web development, database systems, and information system design.

I focus on building practical web-based applications, especially systems that help organize data, improve reporting, and support real-world activities. ## 📌 Featured Project

<div align="center">

### 🕌 Sistem Monitoring Wali Santri Al-Bayan

</div>

A web-based information system developed for **Al-Bayan Islamic Boarding School** to help guardians monitor student activities, attendance, academic progress, memorization, health records, and daily reports.

**Main Features:**

* Admin dashboard
* Student data management
* Attendance management
* Daily activity monitoring
* Guardian monitoring access
* Teacher and musyrif role management
* Web-based reporting system

**Technology Used:**

`PHP Native` `MySQL` `HTML` `CSS` `JavaScript` `Bootstrap`
